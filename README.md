# БИМС (Базовая Информация Миндустри Сервера)
## Что это такое?
Это простая программа на **`Python 3.9`**, которая показывает: 
- **количество игроков** на сервере
- **волна** на сервере
- **карту** на сервере
- **версию ядра** на сервере
- **имя сервера** *без цветовой гаммы*
